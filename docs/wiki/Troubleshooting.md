# Troubleshooting

## Common Problems

- Missing review template: confirm `.github/pull_request_template.md` exists.
- Missing code owners: confirm `.github/CODEOWNERS` exists.
- Workflow not triggering: check Actions permissions and branch rules.
- Discussions unavailable: ensure repository Discussions are enabled in settings.
