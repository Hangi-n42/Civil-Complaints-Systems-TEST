# ADR Index

This directory stores Architecture Decision Records for collaboration and workflow decisions.

## How to Use

1. Copy the template in [template.md](template.md).
2. Create a new ADR file with a sequential number, for example `0001-feature-branch-strategy.md`.
3. Link the decision from the related docs or PR.
